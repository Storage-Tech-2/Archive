Quad bulk hall design has 4 [box display](https://discord.com/channels/1375556143186837695/1454753497638506689/1454753498494144522)s per [slice](https://discord.com/channels/1375556143186837695/1454754736224997386/1454754736858206261) and 4 [box sorter](https://discord.com/channels/1375556143186837695/1454753516269736090/1454753518383792221)s.
## Features
- FHL
- Sticks 2 blocks from both sides
- 4 bulk silos/slice
- Nice smart dropper turns
- [Global](https://discord.com/channels/1375556143186837695/1454753648574992557/1454753649480958066) [box](https://discord.com/channels/1375556143186837695/1454806018348290109/1454806020177133708) placement
- Slice capacity: Top: 1.564 mil Top: 1.594 mil
- Size for 16 slices: 35x39x20
## Considerations
- Lot of inventory prefilling

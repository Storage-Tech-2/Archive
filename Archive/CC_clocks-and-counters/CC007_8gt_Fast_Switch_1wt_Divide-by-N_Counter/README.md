The toggle will switch each time the top rail is pulsed `N` times, where `N` is the number of items in the dropper counter below. Same concept as [CC002 1wt Divide-by-N Counter](https://discord.com/channels/1375556143186837695/1388318562669690982) but can switch faster, at 8gt clock input if needed.
## Features
- Faster switching, works with constant 8gt clock input
## Considerations
- Much larger than other designs, do not use if 8gt clock change or 1wt isn't required

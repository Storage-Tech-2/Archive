Filters and loads items into boxes at 2x hopperspeed. This design is 1 wide AB-tileable, and uses precision box breaking to ensure that output is slice-preserving, meaning that boxes from one slice will not be output at a neighboring slice.
## Features
- 1 wide AB-tileable design
- 2x hopperspeed loading
- Uses precision box breaking for slice preservation
- SSI filters with minimal prefill required
## Considerations
- Items can accumulate in the clocked dropper over time

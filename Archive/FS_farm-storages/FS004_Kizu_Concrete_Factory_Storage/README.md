A basic concrete storage using variable filter loaders.
## Features
- Uses [VIP001 Global Reset Var Loaders](https://discord.com/channels/1375556143186837695/1388318311779139598) to filter items and load boxes. 22 slices. configured for 192k duper.
- Around 1.2 mil capacity in each slice.
- 34 item types: all concrete items plus sand and gravel.
- Sand and gravel have double capacity.
- Partial box collection toggle, which when enabled will collect and store partial boxes in a separate area.
## Considerations
- All overflow is destroyed.
- By default, destroys partial boxes resulting from resetting variable loaders. In general, you should operate the concrete factory for longer than ~12 minutes to get full boxes.

[1.20+] YAMS v2.3 A simple filter-based main storage using components sourced from the Storage Tech archive. There are two versions of this storage, one with a mini bulk, and one with a Full bulk storage.
## Features
- Mini Bulk version: 3,963 hoppers; 100% locked 385,344 bulk capacity/half slice
- Full Bulk version: 5,751 hoppers; 100% locked 1,515,456 bulk capacity/half slice
- Non-stacked empty boxes compatibility
- 6,940 empty box capacity (If using stacked boxes)
- All filters are easily accessible to the player
- Detects chunkloading failure
## Compatibility
[1.20+]

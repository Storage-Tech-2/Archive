Complete Main Storage for 1.21.11
## Features
- Nice Deco
- Color Coding For easier debug
- 80% Hopperlocked
- 2 Chest Halls with 40 slices, 8 chest per slice, 1.5 dc per item
- 64 bulk items
- 20 Mis Categories
- Unstackable Sorter
- Stacked box Storage with /carpet stackableshulkerboxes set to true
- Shulker distribution to all necessary areas
- Basic Protections
- Calibrated sculk hidden doors to access brewer/dye supply area and overflow
- Several Nano Farms
- Villager trading hall with room for 80 villagers
- Several Buffer Box Displays throughout Chest and bulk halls
- Hidden Enchant table + Toggleable nether portal
## Notes
Take care while building if you plan on using this. Several components either require prefill or need some setup for them to function properly (Ex. Shulker crafter). This Storage was built by me for me, it wont be perfect for everyone but having options to choose or take inspiration from is never a bad idea

## Instructions - Basic Sorting
1. Start the chunkloaders, if successful status lamp will light and system will unlock.
2. Put items into input ui, full single type boxes to bulk, mixed boxes in mixed, loose items in mixed shulker.  
3. Hit the break mixed box noteblock if applicable.
4. Hit the start system noteblock
**System will not start unless chunkloading is successful**
## Additional tips
* Hopperlocking for Chest halls can be toggled to let any buffer flow if chests are emptied
* There are signs labeling various components that might not be immediately obvious. As well as some simple instructions.

Built for the needs of a casual singleplayer world, sorting every item in the game including every unstackable at up to 16x hopper speed.
## Features
- 2x 10-Chest halls, each 32 long - 640 items
- Accessible bulk hall, 40 long - 80 items
- 14 MIS categories
- Unstackable sorter
- Copper Golem sorter (24 categories)
- Parallel item unloading
- 94% hopperlocking
- Peripherals (furnace array, brewer, crafting station, shulker crafter, dispenser crafter)
- Fully overflow protected, chunk loading fail check
## Considerations
- The armor sorter can break due to a bug in 1.21.6+. Until Mojang fixes it, armor will go to unstackable overflow.
## Changes from v1
- Replaced Allay sorter with Copper golem sorter, now sorting every unstackable into categories.
- Updated Unstackable sorter, Unloader, Brewer, Dispenser crafter, Furnace array to newer, more relieable versions.
- Added a MIS buffer.
- Added overflow protection and overflow lamps to MIS, MIS buffer, Overflow chests, Unstackable sorter, Unloader buffer.
- Expanded the MIS by 2 categories, item layout updated to 1.21.10.
- Added chunk loading check.
## Compatibility
1.21.9+ (Schematic, WDL: 1.21.10), with armor sorter bug in 1.21.6+

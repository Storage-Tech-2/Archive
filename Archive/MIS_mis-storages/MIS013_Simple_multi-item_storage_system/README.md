A basic multi-item-sorting (MIS) based storage system. An older design that is still useful as a starter storage.
## Features
- Hopperspeed sorting
- Sorts stackable items and unloads the contents of shulker boxes; each column or category (2-wide) can sort up to 54 item types.
- Chunk loader included
- 100% Hopperlocked apart from chunk loaders
## Notes
MIS by Datnerd et al., Chunk loader by Kahyxen

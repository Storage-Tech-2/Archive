CMIS 3.0 is a complete storage system designed for single player or small server use, to sort and store almost every item in 1.20+ within a reasonably compact [user interface](https://discord.com/channels/1375556143186837695/1454754844316274861/1454754845033627671). The system utilizes a combination of Variable Sorting and [Encoded](https://discord.com/channels/1375556143186837695/1454753595424505866/1454753596133605426) Storage, as well as Shulker [Box Sorter](https://discord.com/channels/1375556143186837695/1454753516269736090/1454753518383792221)s for additional [local](https://discord.com/channels/1375556143186837695/1454753648574992557/1454753649480958066) "pseudoBulk" storage of up to 80 different item types. Auto and Manual Restocking logic is included for the (up to) 62 different BULK item types, with parallelized [MIS](https://discord.com/channels/1375556143186837695/1454753725091676293/1454753725829877954) sorting for greater efficiency.
## Features
- Variable Sorting, using [Splitter](https://discord.com/channels/1375556143186837695/1454754768088989789/1454754769674567868)s, Whitelisters, Groupers and [Merger](https://discord.com/channels/1375556143186837695/1454753701204987938/1454753702773526549)s
- Encoded [Bulk Storage](https://discord.com/channels/1375556143186837695/1454753523668484213/1454753524637499577)
- Restocking built-in
- Simple UI
- Remote Bulk (place those laggy hoppers far away from main base!)
## Considerations
- Bulk items will "buffer" up to a full [box](https://discord.com/channels/1375556143186837695/1454806018348290109/1454806020177133708) in the sorting system before sending to storage.
- No first box placement (please see helper doc for recommended usage)

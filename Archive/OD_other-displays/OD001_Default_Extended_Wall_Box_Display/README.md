## Features
- Snow default extended when no box is displayed
- [Global](https://discord.com/channels/1375556143186837695/1454753648574992557/1454753649480958066) first box placement
- Loses last box due to how it's wired
- If global first box dispensing triggers while a box is being replaced, there's a chance the box won'[t](https://discord.com/channels/1375556143186837695/1454753634435858666/1454753635690090539) [be](https://discord.com/channels/1375556143186837695/1454754788473307200/1454754789320560680) collected

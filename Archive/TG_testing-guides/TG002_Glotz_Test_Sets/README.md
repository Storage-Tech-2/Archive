Collection of test item sets made by Glotz. These sets use items leftover from projects in Technical Minecraft servers, covering a real use case for storage systems.
## Test Sets
- Sci Mooshroom set (68,320 items, 104 types): Actual box collection from leftovers after a farm on sci. Note that there are a few renamed items (EG netherrack items renamed to netherrack) which wont be sortable.
- Wavetech combined set (185,676 items, 236 types). Leftovers from miscellaneous projects on the Wavetech server.

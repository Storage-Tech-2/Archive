Sorts item entities at effectively 4gt by separating them into 2 lanes that each get processed at 8gt; two opposing sorters for each item. Item collection is not included.
## Features
- Can sort an item entity every 4gt; up to 1,152,000 items per hour
- Infinitely expandable 2wABt slices
- Works with random inputs
- Compact: Core 7x6x11, Slices 9x10x2

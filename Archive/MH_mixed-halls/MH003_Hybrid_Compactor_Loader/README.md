# Hybrid Compactor Loader

**Tags:** *Functional*

**Original post:** [View on Discord](https://discord.com/channels/1375556143186837695/1493450360906580028)

A hall slice which takes as input compactable items and displays it in three forms. The top chest holds the compactable (eg iron ingots), the middle holds the compacted form (eg iron blocks), and the final holds full boxes of the compacted form (eg boxes of iron blocks). meant to take input from small hopper speed farms around a main storage (not necessarily a recommended practice).
## Features
- works at hopper speed
- 1wt, and fairly compact (7x14)
## Considerations
- If two slices trigger box breaking with incredibly close timings(within 4gt) a full box can be mistakenly sent to overflow
## Compatibility
1.21+

Items are loaded into an accessible shulker box, the subsequent full boxes are then stored in double chests
## Features
- SSI filters
- Low buffer before box replacement
- 200k item capacity per item type
- Overflow box collection
## Considerations
- Average of 3.5 unlocked hoppers per slice
- Displays buffers boxes
## Notes
Please don't lock the hall while it is in operation. Box display from DatNerd, Overflow water stream by Pyraleft (See [WT004 One Tall Output Collection Stream](https://discord.com/channels/1375556143186837695/1455461373415915728))

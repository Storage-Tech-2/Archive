An 8x-speed variable stack separation sorter with hot-switchable box loaders, designed for high-throughput item processing.
## Features
- Compact cuboid exterior design
- Pure shulker box output
- Hopper-locked when powered off
- Timed item refreshing system
- Single-slice hot switching
- Emergency stop (EMS) switch
- 8x speed full-detection box loading
## Considerations
- Processing speed may decrease when handling mixed or excessive item input
- Not ideal box output, can output multiple partial boxes.
## Compatibility
1.19+. There are reports of it breaking in some 1.21.x versions due to bubble column changes.
## Instructions
1. Insert items through the input
2. Single-type shulker boxes will be output
3. Pull down the EMS lever to forcibly stop the machine and eject remaining items

Sorts 1.20 unstackable items at half-hopperspeed throughput. Has two versions: base version sorts 11 unstackables types, while extended version sorts 16, adding cart and boat variants.
## Features
- Sorts at half-hopperspeed (1 item every 16gt)
- Very compact: 6x6x14 for base, 6x8x17 for extended
- Low hopper count only 28 hoppers for base, 46 for extended
- Every clock is independent and pulses only once per item.
- Fully hopper locked versions included where hopper carts are yeeted while locked.
- Includes a fish tank for mobs in a bucket.
## Considerations
- Brushes are not sorted separately, they go into shears output
## Compatibility
1.20+

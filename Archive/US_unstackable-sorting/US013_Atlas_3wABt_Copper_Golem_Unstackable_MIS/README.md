An unstackable sorter using copper golems, with support for multi-item-sorting.
## Features
- 34 hoppers
- Fully hopperlocked
- Overflow Protected
## Considerations
- AB tileable
- Right side of render only has 1.5 dc of space meanwhile left side has 3 dc of space
## Notes
In the schematic there are 8 waxed copper golems per cell, you can reduce the amount to reduce lag, or increase for more speed
## Compatibility
1.21.9+

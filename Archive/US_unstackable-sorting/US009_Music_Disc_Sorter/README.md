A Minecraft device for sorting music discs using hoppers, featuring high-speed sorting with minimal hoppers and no idle carts.
## Features
- Hopper Speed Sorting
- Minimum hoppers
- Idle Cartless
- Extremely Compact, 2 x 8 x 19
- 4w front
## Considerations
- Spawns a Hopper minecart per music disc input
- Not compatible with non-music disc input
## Compatibility
- Built in 1.21.4(On Soontech CMP)

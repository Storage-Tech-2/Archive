A device that sorts all music discs using a combination of signal strength and music duration.
## Features
- Fully hopperlocked
- Hopperspeed sorting for most types with unique signal strengths
- For discs with the same signal strength, sorts between them by measuring disc duration
## Compatibility
Does not sort Lava Chicken and Tears discs so is not compatible with 1.21.7+

Spreadsheets of survival obtainable sortable items for different Minecraft versions.

Made by using the CSV export tool on https://layout.storagecatalog.org/
## Inclusion criteria
- Must be survival obtainable
- Must be either 64 or 16 stackable items (no unstackables)
## Columns
1. Item name
2. ID
3. Max stack size
4. Has block ("Yes" if item is block item)
5. Creative Tabs (semicolon separated list of creative tabs where the item can be found)

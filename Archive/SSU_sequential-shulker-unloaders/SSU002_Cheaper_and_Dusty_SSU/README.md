A Sequential Shulker Unloader optimized to be small and cheap by using more redstone dust than observers. It is inspired by previous designs by Heilz and SamosTheSage.
## Features
- Optimized to be small and cheap
- Each slice filters items at max 1x hopper speed
- The box loader is fully reliable
- Includes an end slice with unloader, mixed loader, and empty box recycling
## Considerations
- Dust may be less lag-friendly than observers, but none of the dust is constantly clocked here, it only activates per box, so there wasn't a noticeable difference in my testing.
## Instructions
1. Expand or trim the tiled version to desired number of slices
2. Set the item filters at ss15 with 4 unstackable blockers like: [63 B B B B] or [15 B B B B]
3. Make sure you have enough storage space for output boxes (full, mixed, and empty)
4. Input mixed boxes and wait for them to process

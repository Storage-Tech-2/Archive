A bulk storage hall that sorts items and packs them into boxes while making sure all items are accessible with a built in box call system - If there is no full box in the hall, you can press the button to break the shulker box in the loader.
## Features
- 3 double chests of box storage per type
- AB tilable SSI SS2 filters
- Precision box loading with box call
- Box call will not output empty boxes
- Default extended box displays with auto first box placement
- Easy to expand, schematic has 32 long hall
- Fully hopperlocked
## Considerations
- Displays do not have empty box collection
## Compatibility
1.16+

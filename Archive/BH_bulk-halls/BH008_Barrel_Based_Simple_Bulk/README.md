A Bulk storage made for easy understanding, scaling (both upwards and horizontally) and using. Features an open cieling design.
## Features
- Hopperspeed Box sorter by acaciachan, so is the input;
- Reachable filters from inside of the bulk via slabs (idea by Obi);
- Expandability upwards and horizontally (you can add layers through the middle, adding 4gt delay for 91 inventory spaces);
- Smart  noteblock default extended display by MoonyInspired and CommandLeo;
- Display collection can be via water streams.
- Display first box dispensing by hitting the noteblock.
- Display chest opens slice if emptied (being refilled).
- Lamp indicators for opened slices on the Deco-ed versions.
- 100% locked*
- Simple, easy to follow wiring.
## Considerations
- While there is no box displayed, its hopper remains unlocked (meaning default 1 hopper per slice is unlocked);
- Less slot density than the traditional bulk layout;
- Slices are opened via dropper latch;
- non-instant global locking with 10bps signal.
## Notes
- The included wiring is Open Ceiling, allowing for a lot of decoration freedom and to fly in and out of the bulk storage;
- The included schematics feature a 959,040 item capacity with shulker boxes in each slice;
- The latches inside of each slice can be set by building the bottom row of the slice, adding hoppers on top and manually dropping the item into the hopper instead of opening the dropper GUI. Only 1 item is needed per latch.

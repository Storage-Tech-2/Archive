Designed for quad displays, with all wiring up top. Hopperlocking is possible with a dustline above the dispenser.

**Features:**
- Automatic box replacement when emptied
- Global first box placement
## Considerations
- Full boxes will buffer in the hopper and dispenser
## Notes
For 1.11+ compatibility replace the glass with slabs/solid blocks and noteblocks with trapdoors/rails.

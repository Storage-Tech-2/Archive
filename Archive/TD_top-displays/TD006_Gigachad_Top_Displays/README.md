**Original post:** [View on Discord](https://discord.com/channels/1375556143186837695/1456439784145813667)

Small top displays that sit directly in the ceiling, allowing for a perfect UI in a quad bulk.
## Features
- Perfect UI
- Global first box placement
- Automatic box replacement
## Considerations
- Dispenser needs to buffer 1 box
- A box may fail to replace if adjacent slice empties in 2 or 3 gt interval
## Notes
Testing: All 4 directions, boxes emptied at 1/2/3/4 gt apart

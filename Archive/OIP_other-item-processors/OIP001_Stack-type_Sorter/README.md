Sorts items based on their stack type at hopperspeed. Has three outputs for unstackable, 16-stackable, and 64 stackable items.
## Features
- Small size: 5x4x2
- No pistons
- 6 hoppers total
## Notes
First dropper should have 38 items in its last slot.

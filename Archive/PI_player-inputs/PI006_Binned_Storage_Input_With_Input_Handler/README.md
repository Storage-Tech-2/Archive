Storage input system based of [PI002 Binned Storage Input](https://discord.com/channels/1375556143186837695/1388318242690564188) which has built in item preprocessing. Has main and bulk inputs; loose items are loaded by a CUD loader; bulk input uses an IsFull so it only outputs full boxes
## Features
- Starts from noteblock
- Pauseable
- Fully hopperlocked (except CUD loader empty box distribution hopper, which should be locked externally)
- CUD loader for loose items
- Very small (7x10x6)
- Good UI

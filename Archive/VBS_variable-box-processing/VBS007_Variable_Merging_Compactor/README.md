Component which takes as input Full boxes of 9-1 compactable items, and crafts and merges them in the same operation at 8x HS. Only outputs full boxes of compacted items.
## Features
- 11x14x6
- Can handle all 9-1 compactable recipes at 8x HS with one device
- Merges while Crafting, maximizing the efficiency of the loading operation
## Considerations
- No included whitelister
- Will Break if non full boxes are input
- Will break if non compactables are input

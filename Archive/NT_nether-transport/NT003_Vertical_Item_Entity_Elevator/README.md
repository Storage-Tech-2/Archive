A device that sends items upwards in entity form, float align items with a boat.
## Features
- Nether compatible
- 20 Blocks per Second
- Minimum cycle time: 9 game ticks (8gt causes torch burnout)
## Considerations
- Requires a Boat
## Compatibility
Works in all game versions.

Splits farm output along multiple ice streams to reduce item entity lifetime and evenly distribute working load.
## Features
- Compatible with 4wt box loaders (ex: [OL003 Globally Clocked 3wt 6x Loader](https://discord.com/channels/1375556143186837695/1389410831594291300))
- Compact
- Nether Compatible
## Considerations
- Requires some minimal prefill (1 item per module)

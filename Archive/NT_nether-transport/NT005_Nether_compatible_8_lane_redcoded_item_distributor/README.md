Distributes item entities evenly among 8 ice lanes. Uses a counter plus a redcoder to route the items so it outputs to a different lane each time the items are sent. Useful for integrating with loading arrays for high output farms.
## Features
- 8 lane output, one lane every 4 blocks on both sides.
- 24gt clockable
- Nether compatible
- Items are output already aligned on edge of block
## Considerations
- A bit expensive due to all the ender chests, you can replace with regular chests but it will be a bit laggier
## Instructions
1. Put items on terracotta block where slime pusher is. make sure items are already 4d aligned for optimal output.
2. Click/power noteblock labelled "clock"

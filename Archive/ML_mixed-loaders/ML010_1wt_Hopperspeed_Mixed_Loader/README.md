1 wide tileable mixed loader which can load multiple item types into boxes at hopperspeed
## Features
- Mixed item compatible
- Hopperspeed
- 1 wide tileable
## Considerations
- Togglestates
- Lots of buffering required to break box
- Very small chance of losing box (around 1 in 50 million)

The top-most hopper will only suck items that have a certain item type at hopperspeed (1 item/8[gt](https://discord.com/channels/1375556143186837695/1454753634435858666/1454753635690090539) or 2.5 items/sec). Two variants are common: Signal Strength 2 ([SS](https://discord.com/channels/1375556143186837695/1454754774833696893/1454754775542267957)2) and Signal Strength 3 (SS3) filters.
## Features
- Basic designs with inventory setup guide
## Considerations
- SS2 filter is not 1 wide tileable, you must A/B tile

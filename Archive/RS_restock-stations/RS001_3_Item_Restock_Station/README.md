A restock station that uses 3 [box display](https://discord.com/channels/1375556143186837695/1454753497638506689/1454753498494144522)s.
## Features
- [Local](https://discord.com/channels/1375556143186837695/1454753648574992557/1454753649480958066) first [box](https://discord.com/channels/1375556143186837695/1454806018348290109/1454806020177133708) placement for both floor and top displays
- Global first box placement can be added for middle display
- Default extended for both floor and top displays
- Collectionless
## Considerations
- Unlocked hoppers
- For the middle display, if 3 adjacent displayed boxes are emptied within 1[gt](https://discord.com/channels/1375556143186837695/1454753634435858666/1454753635690090539) of each other, one of the [slice](https://discord.com/channels/1375556143186837695/1454754736224997386/1454754736858206261)s will fail to replace its box
- Top display use [togglestate](https://discord.com/channels/1375556143186837695/1454754804252282910/1454754804881428533)s

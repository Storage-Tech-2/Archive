This loader will automatically replace the box when items have stopped flowing using a Comparator Update Detector (CUD). Useful for packing mixed items from input into boxes.
## Features
- Hopperspeed input
- Uses the CUD to detect fullness of box, and end of input.
- Supports mixed item input and buffers no items
- Boxes are output as item entities
## Considerations
- It takes a while for the box to break as it must wait until a fader clock runs out.

## Features
- This box loader is designed for farms that only produce one item type (cobble, bamboo..)
- Single slice can be tiled alternating powered and activator rails as shown in the tiled example
- Full boxes get spit as entities from droppers below to follow a water stream to storage
- 3x hopper collection speed per slice
- You can refill  loaders with empty boxes via water stream
## Considerations
- This loader is not fully reliable and could loose a box in 50 million boxes
- Dispenser next to the leafs needs a prefill of one water bucket

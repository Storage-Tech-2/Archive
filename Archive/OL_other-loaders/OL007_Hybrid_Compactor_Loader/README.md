**Original post:** [View on Discord](https://discord.com/channels/1375556143186837695/1493445316136009728)

A device which takes as input a compactable, and displays it in three forms. the top chest holds the compactable, the middle holds the compacted form, and the final holds full boxes of the compacted form. meant to take input from small hopper speed farms around a main storage(not necessarily a recommended practice).
## Features
- works at hopper speed
-1wt, and fairly compact(7x14)
## Considerations
- If two slices trigger box breaking with incredibly close timings(within 4gt) a full box can be mistakenly sent to overflow

A simple 32 furnace array using redcoded instant droppers.
## Features
- Simple & compact size
- Easy to build
- No Entities, low lag
- Runs even on low fuel because fuel is distributed evenly
## Notes
Must use fuel that lasts at least 10s (200 ticks). Keep chunks loaded while running. Includes decorated and non-decorated versions.
## Compatibility
All versions after 1.11. Built with 1.21.10. For older versions, see the video from 3:46.
